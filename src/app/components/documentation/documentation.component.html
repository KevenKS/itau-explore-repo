Documentation Itau Repository Explorer

Propósito

Aplicação desenvolvida para explorar repositórios publicos do itau no GitHub. A pesquisa é feita por linguagem de programação, facilitando a descoberta de projetos e análise de Pull Requests.

Mode de uso

Na pagina inicial você pode buscar por repositórios.

1 - Digite a linguagem de programação de sua preferencia.

2 - Ao clicar em algum repositório você terá acesso aos pull requests que estão abertos e fechados.

3 - Na pagina de pull requests você poderá filtrar entre PRs abertos e fechados.

* Ao clicar no nosso logo você será redirecionado para a pagina inicial. *

Funcionalidades Principais

  • Busca de repositórios por linguagem
  • Visualização de estatísticas (stars, forks, issues)
  • Exploração de Pull Requests
  • Paginação de resultados

Acessibilidade

  • Navegação completa por teclado (Tab, Enter, Espaço)
  • Labels descritivos para leitores de tela
  • Estrutura semântica HTML
  • Contraste adequado de cores

Padrões de Projeto

  • Componentes Standalone Angular
  • Serviços para gerenciamento de estado com Behavior
  • Interfaces tipadas
  • Lazy Loading de rotas
  • Responsividade Mobile First
  • Debounce para otimização de requisições
  • Componentização reutilizável
  • Paginação otimizada
  • Tratamento de estados de erro
  • Clean Code e SOLID principles
